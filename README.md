# Artale RJPQ 跳平台輔助工具

「Artale RJPQ 輔助工具」旨在減輕玩家在跳台挑戰任務中的記憶負擔與溝通成本。透過極簡的版面，提供跨裝置、即時且零阻礙的隊伍雲端連線狀態同步。只要創建房間並將「4 位數房號」分享給隊伍，所有人都能隨時查看目前的開圖進度。

## ✨ 核心特色與功能

- ⚡️ **免登入即時同步**  
  基於 Firebase Realtime Database 實現。無需註冊會員或繁瑣登入，系統將會隨機派發簡潔易記的「4 位數字房號」，達到一秒開房、即刻加入。

- 🖱️ **直覺的防呆標記**  
  - **一般點擊 (左鍵 / 輕觸)**：於該階層平台標上個人隊伍顏色，代表「**安全路徑**」。
  - **標註死路 (右鍵 / 長按)**：將方塊打上紅色大 ✕ 記號，快速警告隊友「**此路不通**」。

- 📱 **為行動裝置打造 (RWD)**  
  不拘泥於電腦端！網格排版在手機面板上會自動變為親自拇指操作的適配模式；重要按鈕在小視窗具備堆疊化與 Touch-Target 標準防誤觸設計。

- 📋 **路線結算與一鍵複製**  
  不再需要費時打字對路線！當一路將 L1 至 L10 皆尋到答案後，日誌追蹤面板會自動將路徑格式化為好讀的文字（例：`12134 21322`），輕點「📋 複製」即可無縫貼上對話框、即時回報！

- ♻️ **活躍防卡分配機制**  
  不佔用免費額度與幽靈資源！為確保所有需要進行挑戰的活躍玩家能夠無阻連線，若網頁被關閉或「超過 30 分鐘完全無動作」，系統將自動解除該裝置連線，並智慧移交管理員給下一位房客，將伺服器保持高效淨空狀態。

## 🚀 玩家入門指南

1. **基本設定**：於首頁輸入能讓隊友辨識的「暱稱 (最多10字)」，並指定一色的隊伍標籤。
2. **開房與入房**：
   - **我是隊長**：點擊「建立新房間 (房首)」，並將右上角自動出現的 `4位數字房號` 或者 `邀請連結` 複製給同伴。
   - **我是隊友**：在「同房號」欄位輸入隊長的 4 碼數字，隨後按下「加入房間 (組員)」。
3. **開始冒險**：按照遊戲內跳板狀況，點擊各個樓層 (L1 到 L10) 記錄路線。所有同房成員都會即時看到您的綠燈或紅叉。
4. **清理再戰**：可以使用底部的工具鈕迅速「清除我的標記」，或是由任一成員點選「清空地圖」。

> **備註**：當全體人員離開房間時，該房間與所有連線紀錄將自動自雲端安全註銷。
